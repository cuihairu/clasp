# Guide

Learn about Clasp's "Cobra-like" usage patterns and best practices.

- `getting-started`: Installation and minimal examples
- `commands`: Command tree, hooks, execution flow
- `flags`: Flag types and parsing semantics
- `completion`: Completion generation and dynamic completion
- `config`: env/config merge and precedence
- `templates`: help/usage/version templates
- `extending`: Custom Value type extensions
